Privacy Policy - ClipForge

Last updated: June 18, 2026

1. Overview

ClipForge is a personal automation tool operated by an individual developer for their own use. It is not a public service and does not have other users.

2. Data Collected

ClipForge processes the following data, exclusively on the developer's own local device:

  • YouTube video URLs and metadata of publicly available videos selected by the developer
  • OAuth access tokens for the developer's own TikTok, Instagram, and YouTube accounts (stored locally only)
  • Generated video clips, transcripts, and configuration files

3. Use of Data

All data is used exclusively to enable the developer to download, edit, and upload videos to their own accounts. No data is shared with third parties, sold, or used for analytics.

4. Data Storage

All data is stored locally on the developer's own computer. Nothing is stored on external servers or cloud services operated by ClipForge.

5. Third-Party Services

ClipForge interacts with the following third-party APIs solely on behalf of the developer's own accounts:

  • YouTube Data API (Google) - to download source videos
  • TikTok Content Posting API - to upload clips to the developer's own TikTok account
  • Instagram Graph API (Meta) - to upload clips to the developer's own Instagram account
  • Anthropic Claude API - for AI-based clip moment selection
